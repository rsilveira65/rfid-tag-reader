# rfid-tag-reader

## Project Setup
```bash
$ npm install
```

## Hardware Setup (PIN/GPIO)
```
SDA --> 24

SCK --> 23

MOSI --> 19

MISO --> 21

GND --> 6

RST --> 22

3.3V --> 1

LED --> 7

LOCK --> 8

BUZZER --> 9
```
